# **App Name**: KickSync

## Core Features:

- Pulsating Kick Recorder: A tactile, 60fps hub featuring a large pulsating log button and a slider for manual entry of kick intensity.
- Firestore Matrix Sync: Ensures instantaneous real-time state synchronization across both devices using a dedicated Firebase database ledger.
- Secure Six-Digit Pairing: An ephemeral code-based tool used to securely establish a permanent link between parental device nodes.
- Collaborative Push Matrix: FCM-powered native push notifications and in-app HUD banners for immediate activity alerts between partners.
- Connectivity Pulse HUD: A continuous system status indicator that displays live connection states to the synchronization server.
- Emotional Reaction Ledger: A shared history timeline that allows partners to react to kick entries with tracked emotional interactions.
- Rhythm Insight Tool: AI-powered tool that analyzes movement patterns and logs to generate a daily summary of rhythmic baby activity.

## Style Guidelines:

- Primary: Vibrant Rose (#FF7A90) for a modern, vital energy that stands out in the glassmorphic interface.
- Secondary Accent: Soft Periwinkle (#A38CF4) used for highlight indicators and reaction elements.
- Background: Gentle Blush (#FFF5F7), providing a warm and clean desaturated base for the frosted elements.
- Headlines and body: 'Poppins', a geometric sans-serif that ensures a precise, high-fashion look appropriate for a modern web app.
- Soft, rounded line icons with a minimalist glassmorphic aesthetic to maintain tactile visual consistency.
- Fixed bottom navigation bar for a thumb-focused smartphone experience, centered around a 60fps hardware-accelerated hub.
- Fluid UI transitions using scale-down (0.94) press transforms and smooth pulsating core recording button effects.
